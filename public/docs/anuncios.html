    # Anuncios

    Listar os anuncios de acordo com o modelo e marca fornecido.

    # Rota

    GET /anuncio/{idMarcaCarro}/{idModeloCarro}/{pagina?}

    # Parâmetros da rota

    Parâmetro Descrição
    idMarcaCarro Id da marca de carro retornado da rota /marca-carro (obrigatório)
    idModeloCarro Id do modelo de carro retornado da rota /modelo-carro/{idMarcaCarro} (obrigatório)
    pagina Página dos anuncios na OLX (opcional, padrão: 1)

    # Response

    Se a requisição for feita com sucesso, o formato da resposta é o seguinte:

    HTTP/1.1 200 OK

[
    {
        "carro": {
            "marca": "fiat",
            "modelo": "uno",
            "quilometragem": 179000,
            "cambio": "Manual",
            "combustivel": "Flex"
        },
        "link": "https://mg.olx.com.br/bh/autos-e-pecas/carros-vans-e-utilitarios/787914461",
        "titulo": "Uno mille,1.0,2006,02 portas, azul escuro",
        "imagem": "https://img.olx.com.br/thumbs256x256/14/148056327247396.jpg",
        "preco": 11500,
        "data": "Hoje 19:20",
        "cidade": "Belo Horizonte - MG"
    }
]

    Se o id da marca for inválido, o formato da resposta é o seguinte:

    HTTP/1.1 422 UNPROCESSABLE ENTITY

{
    "erro": "Marca de carro não encontrada."
}

    Se o id do modelo for inválido, o formato da resposta é o seguinte:

    HTTP/1.1 422 UNPROCESSABLE ENTITY

{
    "erro": "Modelo de carro não encontrado."
}
