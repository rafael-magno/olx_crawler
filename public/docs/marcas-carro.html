    # Marcas de carro

    Listar as marcas de carros disponiveis.

    # Rota

    GET /marca-carro

    # Response

    Se a requisição for feita com sucesso, o formato da resposta é o seguinte:

    HTTP/1.1 200 OK

    {
        "id": "citroen",
        "nome": "Citroen"

        "id": "fiat",
        "nome": "Fiat"
    }
]
