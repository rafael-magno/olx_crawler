    # Modelos de carro

    Listar os modelos de carros disponiveis de uma determinada marca.

    # Rota

    GET /modelo-carro/{idMarcaCarro}

    # Parâmetros da rota

    Parâmetro Descrição
    idMarcaCarro Id da marca de carro retornado da rota /marca-carro (obrigatório)

    # Response

    Se a requisição for feita com sucesso, o formato da resposta é o seguinte:

    HTTP/1.1 200 OK

[
    {
        "marca": "fiat",
        "id": "147",
        "nome": "147"
    },
    {
        "marca": "fiat",
        "id": "uno",
        "nome": "Uno"
    }
]

    Se o id da marca for inválido, o formato da resposta é o seguinte:

    HTTP/1.1 422 UNPROCESSABLE ENTITY

{
    "erro": "Marca de carro não encontrada."
}
